# CodexSwitch Docs Site

This is the standalone Fumadocs documentation site for CodexSwitch.

## Stack

- Next.js
- Fumadocs UI
- Fumadocs MDX
- Tailwind CSS
- Orama search

## Local development

```powershell
npm install
npm run dev
```

Open:

- `http://localhost:3000/zh`
- `http://localhost:3000/en`
- `http://localhost:3000/zh/docs`
- `http://localhost:3000/en/docs`

## Production

```powershell
npm run build
npm run start
```

The site is configured for a normal Next.js server deployment, not static export.
