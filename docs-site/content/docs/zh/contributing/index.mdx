欢迎提交 issues 和 pull requests。涉及路由、协议转换、价格、用量统计或配置写入的改动，应在 `CodexSwitch.Tests` 中补充聚焦测试。

提交 PR 前建议运行：

```powershell
dotnet test CodexSwitch.Tests/CodexSwitch.Tests.csproj -c Release
./build/Validate-Changelog.ps1
```
