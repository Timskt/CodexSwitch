---
title: 测试策略
description: 路由、适配器、价格、用量、配置和 UI 应该在哪里加测试。
icon: TestTube2
---

修改共享行为时添加聚焦测试。当前覆盖范围包括：

- 配置写入器。
- Provider 路由。
- Responses payload 构建。
- OpenAI Responses 适配器行为。
- OpenAI Chat 适配器行为。
- Anthropic Messages 适配器行为。
- 用量解析和归一化。
- 价格计算。
- i18n 服务。
- 更新检查。
- UI 组件结构和状态。

开发时可先运行相关测试类，合并前再运行完整测试项目。
