根解决方案包含：

| 路径 | 用途 |
| --- | --- |
| `CodexSwitch/` | 主 .NET 10 Avalonia 桌面应用和本地代理。 |
| `CodexSwitch.Tests/` | 覆盖路由、价格、配置写入、用量、i18n 和更新的 xUnit 测试。 |
| `CodexSwitchUI/` | UI 组件 submodule，包含 Avalonia 控件和 ECharts 支持。 |
| `build/` | 发布和验证脚本。 |
| `docs/` | 现有发布流程说明和支持 fixture。 |
| `docs-site/` | 当前 Fumadocs Web 文档站。 |

`MainWindowViewModel` 是应用的中心协调器，负责配置、provider 状态、代理生命周期、用量仪表盘、价格、更新、OAuth 登录、开机启动和 UI 命令。
