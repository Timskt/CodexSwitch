## Restore

```powershell
dotnet restore CodexSwitch.Tests/CodexSwitch.Tests.csproj
```

## Build

```powershell
dotnet build CodexSwitch.Tests/CodexSwitch.Tests.csproj -c Release --no-restore
```

## Test

```powershell
dotnet test CodexSwitch.Tests/CodexSwitch.Tests.csproj -c Release --no-build --no-restore
```

## 单个测试类

```powershell
dotnet test CodexSwitch.Tests/CodexSwitch.Tests.csproj -c Release --filter FullyQualifiedName~PriceCalculatorTests
```

## 验证 changelog

```powershell
pwsh ./build/Validate-Changelog.ps1
```
