| 方法 | 路由 | 说明 |
| --- | --- | --- |
| `GET` | `/health` | 代理健康检查。 |
| `GET` | `/v1/models` | Responses 风格模型发现。 |
| `POST` | `/v1/responses` | 主要 Codex 入站路由。 |
| `POST` | `/v1/messages` | 支持范围内的 Claude Code / Anthropic 风格入站路由。 |

CodexSwitch 当前暴露 Responses 风格入站路由，不把 `/v1/chat/completions` 作为公开入站 API。
