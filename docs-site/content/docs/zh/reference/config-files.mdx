---
title: 配置文件
description: CodexSwitch 读取或写入的本地文件。
icon: Files
---

在 Windows 上，主要文件包括：

| 文件 | 用途 |
| --- | --- |
| `%APPDATA%\CodexSwitch\config.json` | Providers、路由、UI、代理、OAuth 元数据和本地设置。 |
| `%APPDATA%\CodexSwitch\model-pricing.json` | 可编辑价格目录。 |
| `%APPDATA%\CodexSwitch\usage-logs\yyyy\MM\usage-yyyy-MM-dd.jsonl` | 按日期分片的本地请求用量日志。 |
| `%APPDATA%\CodexSwitch\icons\` | 缓存的 provider 和模型图标。 |
| `%USERPROFILE%\.codex\config.toml` | 代理启用时的受管理 Codex 配置。 |
| `%USERPROFILE%\.codex\auth.json` | 受管理 Codex auth，除非启用 auth preservation。 |
| `%USERPROFILE%\.claude\settings.json` | 受管理 Claude Code 设置。 |

这些配置文件可能包含 API key 和 OAuth token，应按敏感文件处理。
