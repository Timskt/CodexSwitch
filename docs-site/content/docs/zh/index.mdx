CodexSwitch 会把 Codex 变成一个本地、可切换、可观察的 AI Provider 工作台。它在本机运行兼容 OpenAI Responses 的代理服务，写入受管理的客户端配置，把模型请求路由到不同上游，在必要时转换协议，并记录本地用量和成本估算。
