Feedback 组件让后台操作变得可见：provider 状态、代理状态、OAuth 进度、更新检查、空结果、加载和用量刷新。

## 审查状态

- Success、warning、error 和 neutral 变体。
- Alert anatomy：icon、title、description、rich content、action slot 和默认 glyph 行为。
- Sonner service lifecycle：show、dismiss、clear、visible stack、toast limit trimming、loading duration、close-button visibility 和 rich-color viewport 状态。
- Badge anatomy 和 interaction：variant、status dot、inline icon content、spinner content、link-like badge、command activation、focus-visible、keyboard activation，以及 `Activated` source metadata。
- Avatar 和 AvatarGroup anatomy / interaction：image loading status、delayed fallback、error fallback、status badge、重叠堆叠、继承尺寸、隐藏成员和 AvatarGroupCount 溢出成员。
- 长文本换行。
- 加载和空状态。
- Empty state 的主/次操作，包括 loading、disabled 和 command-blocked 触发抑制，并暴露可见 action state class。
- Reduced-motion 预期。
