表单组件覆盖主要 provider、model、settings 和 auth 工作流。

## 组件族

- Button
- Button Group：相关动作分组、连接态 item anatomy、separator、text、嵌套组和 input 组合
- Split Button：主命令加 Web 风格主指针 release 与 Enter/Space/ArrowDown menu trigger state、带 source metadata 的 `OpenChanged`、close-on-select action row、loading suppression 和焦点返回
- Input Group：root border/focus ring anatomy、inline/block addon、动作按钮、快捷键提示和 textarea footer
- Input OTP：分组验证码 slot、slot anatomy、separator、paste 输入、pattern 过滤、invalid state 和键盘导航
- Label：关联目标控件、required marker anatomy、access key、intent styling 和 disabled target 样式
- Field 标签、描述、必填状态、消息、横向/响应式 orientation、FieldGroup、FieldSet、FieldContent、FieldSeparator 和 FieldError
- TextBox
- Select：trigger、selected value、placeholder、popup 动画、disabled item、带 source metadata 且与 Web `onValueChange` 对齐的 `ValueChanged`，以及带 source metadata 且与 Web `onOpenChange` 对齐的 `OpenChanged`
- Combobox：可过滤的预定义选择、主指针 release trigger 对齐、带 source metadata 的 popup 打开/关闭动画、高亮项/选中项、`SelectionChanged` old/new item/index/value/source metadata、清除动作、loading/empty slot，以及 Arrow、Enter、Escape 行为
- Native Select：option value、optgroup label、disabled option、invalid state、原生选择行为、带 source metadata 的 `ValueChanged` old/new option value，以及带 source metadata 的 `OpenChanged` popup state
- Calendar：单日期、日期范围、outside day、week number、disabled bounds、command-blocked day button、键盘导航，以及带 source metadata 的 `SelectedDateChanged`、`RangeChanged`、`DisplayDateChanged` 和 `ActiveDateChanged`
- Date Picker：Popover + Calendar 组合、主指针 release trigger 对齐、单选和范围选择、清除动作、打开/关闭动画、Escape 关闭、loading/disabled guard，以及带 source metadata 的 open/select/range event
- Checkbox：label-target anatomy、checked/unchecked/indeterminate state class、与 Web `onCheckedChange` 对齐且带 source metadata 的 `CheckedStateChanged`、disabled guard、intent，以及 Field/FieldSet 组合
- Radio 和 Radio Group：root-owned value、item value、主指针 release item activation、`ValueChanged` old/new item/index/value/source metadata、orientation、loop、required、disabled 与 loading suppression
- Switch 和独立 Toggle：用于两态控件，并补齐带 source metadata 的 `CheckedChanged` / `PressedChanged` Web 事件对齐；Toggle Group 作为独立页面覆盖 single/multiple value、source-aware `ValueChanged`、默认 `spacing=2` 间距、`spacing=0` 连接态、垂直方向、disabled item 和 roving keyboard focus
- Slider：token 化 track/thumb 状态、hover/pressed/dragging 动画、与 Web `onValueChange` 对齐的 `ValueChanging`，以及与 Web `onValueCommit` 对齐的 `ValueCommitted`

Field 现在覆盖当前 shadcn Field 契约：grouped fieldset、legend、responsive field group、content/title/description slot、inline separator、invalid state，以及单条或多条 error message，Docs 中都有独立 AXAML 案例并可在渲染案例下方展开源码。Button Group、Input Group、Input OTP 和 Label 现在也分别有独立 anatomy 案例，因此 connected item shape、addon alignment、grouped slot、separator、target association、required marker 与 disabled-target styling 不再藏在 composition 示例里。Select 现在覆盖 default、state、anatomy 和 interaction 案例，每个渲染案例下方都会展开自己的本地 AXAML，并展示 pointer/keyboard/programmatic value/open source metadata。Native Select 现在拆成 state、anatomy、composition 和 interaction 案例：anatomy 会单独展示 trigger、selected content、chevron、popup surface、option value、optgroup label、disabled option、invalid compact trigger，并继续保留明确且 source-aware 的 value/open event 路径。Combobox interaction 现在也展示 pointer、keyboard、input、focus、clear、item 和 programmatic 路径的 source-aware OpenChanged，并保留 selection metadata。Calendar 现在拆成 state、anatomy、composition 和 interaction 案例：anatomy 会单独展示 header、weekday grid、outside day、range edge、week number 和 bounded day cell；interaction 则通过 selected-date、range、active-date、display-date change event 和 command-blocked day activation 暴露带 pointer/keyboard/programmatic source metadata 的 DayPicker 风格 `onSelect` 与 `onMonthChange` 对齐路径。Date Picker 也会把同样的 source metadata 传递到 popover open/close、Calendar selection sync、range completion 和 keyboard clear 路径。Switch、Checkbox 和独立 Toggle 的 interaction 案例现在会展示带 pointer/keyboard/programmatic source metadata 的实时 checked/pressed 变化，同时继续把本地 AXAML 展开源码放在每个渲染组件下方。Toggle Group 也已经从独立 Toggle 中拆出，group 的 state、anatomy 和 interaction 都在渲染案例下方提供对应 AXAML 展开源码，并展示 pointer/keyboard/programmatic value-change source metadata。Slider 现在补齐 value-change/value-commit 事件预览、键盘 commit 行为、垂直方向、disabled guard 和每个案例下方的 AXAML 展开源码。
