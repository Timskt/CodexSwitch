`CodexSwitchUI` submodule 包含可复用 Avalonia 控件、tokens、themes 和已有桌面组件画廊。

组件画廊的分类会作为本文档分区的来源：

- Overview
- Tokens
- Layout
- Forms
- Navigation
- Overlay
- Feedback
- Data Display
- Utilities
- Design Review

这个 Fumadocs 文档站不会替代 Avalonia 组件画廊，而是从概念和贡献者视角说明设计系统。

Layout 覆盖现在包含 shadcn 风格的 Sidebar Provider、Sidebar、Trigger、Rail、Inset 契约，以及 Resizable panel、handle、grip、track、orientation 和 constraint anatomy。Sidebar trigger 和 rail command 会在改变 open state 前同步 `can-toggle` 与 `command-blocked`，并且 Avalonia Docs 画廊里每个案例都会保留自己的内联 AXAML 源码展开流程。

Primitive 覆盖现在包含 Direction provider：把 Web 的 `ltr` / `rtl` 状态映射到 Avalonia `FlowDirection`，说明嵌套方向岛，并且每个渲染案例都保留自己的内联 AXAML 源码展开按钮。
