`CodexSwitchUI` submodule 包含可复用 Avalonia 控件、tokens、themes 和已有桌面组件画廊。

组件画廊的分类会作为本文档分区的来源：

- Overview
- Tokens
- Forms
- Navigation
- Overlay
- Feedback
- Data Display
- Utilities
- Design Review

这个 Fumadocs 文档站不会替代 Avalonia 组件画廊，而是从概念和贡献者视角说明设计系统。
