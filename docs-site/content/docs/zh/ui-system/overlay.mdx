Overlay 组件用于 provider 编辑器、危险确认 alert dialog、筛选 sheet、Vaul 风格 drawer、command dialog、popover、hover 预览和焦点管理。

## 审查状态

- 打开和关闭。
- Dialog root/trigger/overlay/content/close 组合、带 source metadata 的 `OpenChanged`、modal scrim、Enter/Space trigger toggle、Escape/outside 关闭和焦点返回。
- Alert dialog 的 trigger/content 组合、带 source metadata 的 `OpenChanged`、cancel/action 路径、最小破坏性焦点、modal scrim 和 required-response outside pointer 策略。
- Sheet trigger/content 组合、继承 source-aware `OpenChanged`、modal scrim、right/left/top/bottom 侧边位置、边缘滑入/滑出动画、close control 和焦点返回。
- Drawer trigger/content 组合、继承 source-aware `OpenChanged`、modal scrim、方向、handle 拖拽阈值/释放、drag-dismiss-ready 状态、可滚动内容区、固定 footer actions、关闭控件和焦点返回。
- Command dialog trigger/content 组合、继承 source-aware `OpenChanged`、input forwarding、搜索过滤、separator、shortcut、close-on-select、loading suppression、带 source metadata 的 item-selected event 和焦点返回。
- Popover trigger/content/close/arrow 组合、带 source metadata 的 `OpenChanged`、Enter/Space trigger toggle、side/align 动画、Escape/outside 关闭和焦点返回。
- Tooltip provider/trigger/content 组合、hover 延迟、focus 立即打开、带 source metadata 的 `OpenChanged`、Escape/Enter/Space 关闭、side-aware arrow、快捷键内容和 disabled trigger wrapper。
- Hover card trigger/content 组合、open-change event、打开/关闭延迟、side/align 位置、arrow 可见性、富内容预览和 disabled-root suppression。
- Focus trap。
- Escape、outside-click、close-on-select、drag-release 和焦点返回行为。
- 危险操作确认文案。
- 明暗主题对比度。
