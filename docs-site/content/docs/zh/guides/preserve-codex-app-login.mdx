---
title: 保留 Codex App 登录
description: 在使用本地代理时继续保留依赖 ChatGPT 登录的 Codex App 插件能力。
icon: ShieldCheck
---

如果需要 Codex App 插件，先在 Codex App 中用 ChatGPT 登录，然后在应用代理配置前启用登录保留设置。

## 设置项

```text
Settings > Auth > Preserve Codex App ChatGPT login for plugins
```

## 运行行为

该模式下，CodexSwitch 仍会写入 `config.toml`，但会保留原来的 `auth.json` 登录状态。

当插件能力依赖原始 ChatGPT 登录，同时又希望 Codex 流量经过本地代理时，使用该模式。
