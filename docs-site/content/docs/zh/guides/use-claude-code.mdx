---
title: 使用 Claude Code
description: 通过 CodexSwitch 受管理配置让 Claude Code 走本地代理。
icon: SquareTerminal
---

CodexSwitch 可以写入受管理的 Claude Code 设置，让 Claude Code 连接本地代理。

## 受管理文件

```text
%USERPROFILE%\.claude\settings.json
```

## 设置路径

1. 配置 Claude 兼容或 Anthropic Messages provider。
2. 为 Claude Code 选择活动 provider。
3. 保持本地代理启用。
4. 让 CodexSwitch 写入受管理的 Claude Code 设置。
5. 验证 Claude Code 可以访问本地 base URL 和模型。

## 恢复行为

受管理配置启用时，原始设置保存为 `settings.json.bak`；代理停止时恢复原文件。
