当上游服务暴露 `/chat/completions`，但本地客户端仍向 CodexSwitch 发送 Responses 风格请求时，使用 `OpenAiChat`。

## 转换行为

CodexSwitch 会把入站 Responses 请求转换为 Chat Completions payload，转发到上游，再把结果转换回 Responses 形态。

## 快速配置

1. 创建或选择 Chat 兼容 provider。
2. 将协议设置为 `OpenAiChat`。
3. 填入 base URL 和 API key。
4. 将本地模型名映射到上游 chat 模型名。
5. 先发送简单请求，再测试工具调用较多的 prompt。

## 故障排查

如果流式形态或工具调用不符合预期，对照 [协议](/zh/docs/core-concepts/protocols) 检查上游 Chat Completions 行为。
