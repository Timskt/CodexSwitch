title: 配置 OpenAI Responses Provider
description: 接入已经支持 OpenAI Responses API 的上游服务。

---

当上游服务可以直接接收 Responses 风格请求时，使用 `OpenAiResponses`。

## 快速配置

1. 从内置 OpenAI Responses 模板添加 provider，或创建自定义 provider。
2. 将协议设置为 `OpenAiResponses`。
3. 填入上游 base URL 和 API key。
4. 配置默认模型和需要的上游模型映射。
5. 将 provider 设置为活动 provider。

## 验证

```text
GET http://127.0.0.1:12785/v1/models
POST http://127.0.0.1:12785/v1/responses
```

如果 `/v1/models` 正常但 `/v1/responses` 失败，优先检查模型映射、service tier 覆盖和上游认证。
