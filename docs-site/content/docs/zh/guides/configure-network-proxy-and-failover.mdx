网络行为由本地绑定、出站 provider 访问、provider 可用性和模型回退规则共同决定。

## 默认值

- 保持本地代理 host 为 `127.0.0.1`。
- 使用 provider 各自的 base URL 访问上游。
- 先确认 `/health`，再排查上游 provider。
- 先确认 `/v1/models`，再排查请求转换。

## 回退检查

1. 确认活动 provider 支持请求模型。
2. 确认回退 provider 支持同一模型或别名。
3. 确认被禁用的 provider 没有被预期接收流量。
4. 先检查 provider 认证，再判断路由是否错误。
