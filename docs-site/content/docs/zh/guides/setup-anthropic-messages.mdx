当上游服务接收 Anthropic Messages 风格请求时，使用 `AnthropicMessages`。

## 快速配置

1. 创建或选择 Anthropic Messages provider。
2. 将协议设置为 `AnthropicMessages`。
3. 填入 provider API key。
4. 配置默认 Claude 兼容模型。
5. 先测试基础 prompt，再测试工具调用或 reasoning 行为。

## CodexSwitch 会映射什么

- Message payload。
- 支持范围内的 thinking 或 reasoning 字段。
- 支持范围内的工具调用结构。
- 用于本地日志的用量归一化。

Claude Code 工作流继续阅读 [使用 Claude Code](/zh/docs/guides/use-claude-code)。
