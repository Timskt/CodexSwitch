Codex OAuth provider 使用专门的登录流程连接 ChatGPT Codex 后端。适合希望通过已认证 Codex 后端路由，而不是普通 API key provider 的场景。

## 快速配置

1. 选择 Codex OAuth provider。
2. 在桌面 UI 中点击 Codex OAuth 登录。
3. 完成浏览器登录流程。
4. 确认账号出现在 provider 状态中。
5. 将该 provider 设置为活动 provider。

## 多账号行为

CodexSwitch 会在本地保存 OAuth 账号元数据。应用配置目录应按敏感目录处理，因为 OAuth 状态和 API key 一样重要。
