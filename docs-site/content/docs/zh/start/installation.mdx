CodexSwitch 的发布包是自包含 Native AOT 产物。发布产物可在 [GitHub Releases](https://github.com/AIDotNet/CodexSwitch/releases) 下载。

## Release 产物

| 平台 | 产物 |
| --- | --- |
| Windows x64 | `CodexSwitch-vX.Y.Z-win-x64-setup.exe` |
| Linux x64 | `CodexSwitch-vX.Y.Z-linux-x64.AppImage` |
| macOS arm64 | `CodexSwitch-vX.Y.Z-osx-arm64.dmg` |

## 安装后验证

1. 启动 CodexSwitch。
2. 确认代理状态已启用。
3. 确认本地端点是 `http://127.0.0.1:12785/v1`。
4. 打开 [快速开始](/zh/docs/start/quick-start)，激活一个 provider。

## macOS 提示

未签名或临时签名的构建可能被 Gatekeeper 阻止。签名和 notarization 所需的可选密钥见 [macOS 签名与公证](/zh/docs/operations/macos-signing-and-notarization)。
