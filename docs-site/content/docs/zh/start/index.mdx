Start 分区对应应用的首次使用路径：安装、激活 provider、保持本地代理开启，然后让 Codex 使用本地端点。

## 推荐路径

1. 阅读 [安装](/zh/docs/start/installation)，选择对应平台产物。
2. 按 [快速开始](/zh/docs/start/quick-start) 激活第一个 provider。
3. 如果要开发或调试，阅读 [源码运行](/zh/docs/start/run-from-source)。
4. 如果只知道问题类型，打开 [文档目录](/zh/docs/start/docs-directory) 快速定位。

## 默认本地端点

```text
http://127.0.0.1:12785/v1
```
