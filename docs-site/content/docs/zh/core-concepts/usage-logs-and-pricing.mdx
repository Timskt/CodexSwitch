CodexSwitch 会在本地记录请求用量，并根据可编辑的模型价格目录估算成本。

## 本地文件

| 文件 | 用途 |
| --- | --- |
| `%APPDATA%\\CodexSwitch\\model-pricing.json` | 用于估算成本的可编辑价格目录。 |
| `%APPDATA%\\CodexSwitch\\usage-logs\\yyyy\\MM\\usage-yyyy-MM-dd.jsonl` | 按日期分片的用量日志。 |

## 仪表盘视图

用量仪表盘包含请求日志、provider 统计、模型统计、24 小时、7 天和 30 天趋势。

成本依赖本地价格目录，应视为估算值，而不是正式账单。
