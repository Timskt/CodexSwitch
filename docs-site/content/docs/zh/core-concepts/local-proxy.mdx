`ProxyHostService` 负责本地 Kestrel 服务。它绑定配置中的 host/port，映射本地路由，在运行时应用受管理的 Codex 和 Claude Code 配置，并在停止或禁用时恢复备份。

## 本地路由

| 路由 | 用途 |
| --- | --- |
| `/health` | 本地代理健康检查。 |
| `/v1/models` | 返回当前 provider 可见的模型。 |
| `/v1/responses` | 主要的 OpenAI Responses 风格入口。 |
| `/v1/messages` | Claude Code / Anthropic 风格入口，取决于支持范围。 |

## 运行行为

- 除非明确要对外暴露，否则保持 host 为 `127.0.0.1`。
- Provider 解析按请求发生。
- 用量会在响应可归一化后写入日志。
- 代理禁用时恢复受管理文件的原始版本。
