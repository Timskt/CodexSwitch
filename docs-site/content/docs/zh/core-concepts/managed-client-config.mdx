受管理客户端配置是可恢复的。代理启用时，CodexSwitch 写入本地客户端配置，并用 `.bak` 备份保存原文件。

## 受管理文件

| 文件 | 用途 |
| --- | --- |
| `%USERPROFILE%\\.codex\\config.toml` | 受管理的 Codex 端点配置。 |
| `%USERPROFILE%\\.codex\\auth.json` | 受管理的 Codex auth，除非启用 Codex App 登录保留。 |
| `%USERPROFILE%\\.claude\\settings.json` | 受管理的 Claude Code base URL、token 和模型设置。 |

## 安全规则

这些文件可能包含 API key、OAuth token 和本地路由状态，应按敏感文件处理。
