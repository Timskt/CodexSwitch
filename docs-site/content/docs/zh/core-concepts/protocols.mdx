---
title: 协议
description: Responses、Chat Completions 和 Anthropic Messages 适配器。
icon: Cable
---

CodexSwitch 对外暴露 Responses 风格入口，并把出站请求适配成 provider 对应协议。

| Provider 协议 | 上游端点 | 说明 |
| --- | --- | --- |
| `OpenAiResponses` | `/responses` | 直接转发 Responses 请求，并处理模型、service tier 和请求覆盖。 |
| `OpenAiChat` | `/chat/completions` | 把入站 Responses 请求转换为 Chat Completions，再把结果转换回 Responses 形态。 |
| `AnthropicMessages` | `/messages` | 在支持范围内把 Responses 或 Messages 流量转换为 Anthropic 兼容 payload。 |

CodexSwitch 当前不把 `/v1/chat/completions` 暴露为公开入站 API。
