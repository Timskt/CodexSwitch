Tagged releases 始终把 macOS 产物打包成 `.dmg` 内的 `.app` bundle。Developer ID 签名和 Apple notarization 是可选项。

## 可选 secrets

| Secret | 用途 |
| --- | --- |
| `MACOS_CERTIFICATE_BASE64` | base64 编码的 `.p12` Developer ID Application 证书。 |
| `MACOS_CERTIFICATE_PASSWORD` | `.p12` 密码。 |
| `MACOS_KEYCHAIN_PASSWORD` | CI 临时 keychain 密码。 |
| `MACOS_SIGNING_IDENTITY` | 可选的显式 Developer ID identity。 |
| `MACOS_NOTARY_APPLE_ID` | 用于 notarization 的 Apple ID。 |
| `MACOS_NOTARY_TEAM_ID` | Apple Developer Team ID。 |
| `MACOS_NOTARY_PASSWORD` | Apple ID app-specific password。 |

如果没有这些 secrets，CI 仍会构建 ad-hoc signed DMG，但下载后的产物可能被 Gatekeeper 阻止。
