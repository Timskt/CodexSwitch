CodexSwitch 可以在桌面应用内检查 GitHub Releases，并下载适合当前系统的安装包。

## 运行行为

- Release 元数据来自仓库 release feed。
- 产物选择取决于平台和架构。
- 下载对象是安装产物，而不是源码归档。

## 故障排查

如果更新检查失败，确认能访问 GitHub Releases，并检查目标 release 是否包含 [发布指南](/zh/docs/operations/release-guide) 中说明的产物名称。
