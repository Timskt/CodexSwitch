CodexSwitch 设计为本地工作台工具。除非明确要对外暴露，否则保持代理绑定到 `127.0.0.1`。

## 敏感数据

以下内容应按敏感数据处理：

- Provider API key。
- OAuth token 和账号元数据。
- `%APPDATA%\CodexSwitch\config.json`。
- `%USERPROFILE%\.codex\auth.json`。
- `%USERPROFILE%\.claude\settings.json`。
- 形似 auth 文件的测试 fixture。

## 暴露规则

除非你已经有单独的访问控制方案，否则不要把代理绑定到公网接口。本地代理的目标用户是本机 Codex 和 Claude Code 客户端。
