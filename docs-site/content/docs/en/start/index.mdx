Start pages follow the same first-run shape as the app itself: install, activate a provider, keep the local proxy enabled, then let Codex use the local endpoint.

## Recommended path

1. Read [Installation](/en/docs/start/installation) to choose the right artifact.
2. Follow [Quick Start](/en/docs/start/quick-start) to activate the first provider.
3. Use [Run From Source](/en/docs/start/run-from-source) if you are developing locally.
4. Open [Docs Directory](/en/docs/start/docs-directory) when you know the problem area but not the exact page.

## Default local endpoint

```text
http://127.0.0.1:12785/v1
```
