CodexSwitch release builds are self-contained Native AOT packages. Download published builds from [GitHub Releases](https://github.com/AIDotNet/CodexSwitch/releases) when release artifacts are available.

## Release artifacts

| Platform | Artifact |
| --- | --- |
| Windows x64 | `CodexSwitch-vX.Y.Z-win-x64-setup.exe` |
| Linux x64 | `CodexSwitch-vX.Y.Z-linux-x64.AppImage` |
| macOS arm64 | `CodexSwitch-vX.Y.Z-osx-arm64.dmg` |

## Verify after install

1. Launch CodexSwitch.
2. Confirm the proxy status is enabled.
3. Confirm the local endpoint is `http://127.0.0.1:12785/v1`.
4. Open [Quick Start](/en/docs/start/quick-start) and activate a provider.

## macOS note

Unsigned or ad-hoc signed builds can be blocked by Gatekeeper. Signed and notarized DMGs require the optional release secrets documented in [macOS Signing And Notarization](/en/docs/operations/macos-signing-and-notarization).
