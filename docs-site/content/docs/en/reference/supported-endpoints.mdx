---
title: Supported Endpoints
description: Local routes exposed by the CodexSwitch proxy.
icon: Waypoints
---

| Method | Route | Description |
| --- | --- | --- |
| `GET` | `/health` | Proxy health check. |
| `GET` | `/v1/models` | Responses-style model discovery. |
| `POST` | `/v1/responses` | Main Codex inbound route. |
| `POST` | `/v1/messages` | Claude Code / Anthropic-style inbound route where supported. |

CodexSwitch currently exposes Responses-style inbound routes. It does not expose `/v1/chat/completions` as a public inbound API.
