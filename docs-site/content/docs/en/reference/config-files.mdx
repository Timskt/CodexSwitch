---
title: Config Files
description: Local files written or read by CodexSwitch.
icon: Files
---

On Windows, the main files are:

| File | Purpose |
| --- | --- |
| `%APPDATA%\CodexSwitch\config.json` | Providers, routing, UI, proxy, OAuth metadata, and local settings. |
| `%APPDATA%\CodexSwitch\model-pricing.json` | Editable pricing catalog. |
| `%APPDATA%\CodexSwitch\usage-logs\yyyy\MM\usage-yyyy-MM-dd.jsonl` | Partitioned local request usage logs. |
| `%APPDATA%\CodexSwitch\icons\` | Cached provider and model icons. |
| `%USERPROFILE%\.codex\config.toml` | Managed Codex config while proxy is enabled. |
| `%USERPROFILE%\.codex\auth.json` | Managed Codex auth unless auth preservation is enabled. |
| `%USERPROFILE%\.claude\settings.json` | Managed Claude Code settings. |

Treat config files as sensitive because they can contain API keys and OAuth tokens.
