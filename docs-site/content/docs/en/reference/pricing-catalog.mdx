---
title: Pricing Catalog
description: Local pricing data used for usage cost estimates.
icon: BadgeDollarSign
---

The model pricing catalog lives beside application config:

```text
%APPDATA%\CodexSwitch\model-pricing.json
```

It supports tiered pricing, cache-read pricing, Claude cache-creation billing, and fast-mode multipliers.

Cost estimates depend on this local catalog and should not be treated as billing statements.
