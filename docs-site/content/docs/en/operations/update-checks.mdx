CodexSwitch can check GitHub Releases from the desktop app and download the installer that matches the current system.

## Runtime behavior

- Release metadata comes from the repository release feed.
- The selected artifact depends on platform and architecture.
- Downloads are installer artifacts, not source archives.

## Troubleshooting

If update checks fail, confirm network access to GitHub Releases and verify that the target release includes the expected artifact name from [Release Guide](/en/docs/operations/release-guide).
