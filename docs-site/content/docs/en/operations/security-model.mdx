CodexSwitch is designed as a local workstation tool. Keep the proxy on `127.0.0.1` unless you intentionally want to expose it beyond your machine.

## Sensitive data

Treat these as sensitive:

- Provider API keys.
- OAuth tokens and account metadata.
- `%APPDATA%\CodexSwitch\config.json`.
- `%USERPROFILE%\.codex\auth.json`.
- `%USERPROFILE%\.claude\settings.json`.
- Test fixtures that resemble auth files.

## Exposure rule

Do not bind the proxy to a public interface unless you have a separate access-control story. The local proxy is meant to serve local Codex and Claude Code clients.
