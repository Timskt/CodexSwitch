Issues and pull requests are welcome. For changes that affect routing, protocol conversion, pricing, usage accounting, or config writing, include focused tests in `CodexSwitch.Tests`.

Before opening a pull request:

```powershell
dotnet test CodexSwitch.Tests/CodexSwitch.Tests.csproj -c Release
./build/Validate-Changelog.ps1
```
