The root solution includes:

| Path | Purpose |
| --- | --- |
| `CodexSwitch/` | Main .NET 10 Avalonia desktop app and local proxy. |
| `CodexSwitch.Tests/` | xUnit coverage for routing, pricing, config writing, usage, i18n, and updates. |
| `CodexSwitchUI/` | UI component submodule with Avalonia controls and ECharts support. |
| `build/` | Release and validation scripts. |
| `docs/` | Existing release process notes and support fixtures. |
| `docs-site/` | This Fumadocs web documentation site. |

`MainWindowViewModel` is the central application coordinator for app configuration, provider state, proxy lifecycle, usage dashboard state, pricing, updates, OAuth login, startup registration, and UI commands.
