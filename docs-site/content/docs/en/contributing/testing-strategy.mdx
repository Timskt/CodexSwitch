Add focused tests when changing shared behavior. Existing coverage areas include:

- Config writers.
- Provider routing.
- Responses payload building.
- OpenAI Responses adapter behavior.
- OpenAI Chat adapter behavior.
- Anthropic Messages adapter behavior.
- Usage parsing and normalization.
- Price calculation.
- i18n services.
- Update checks.
- UI component structure and states.

Run the relevant class directly while iterating, then run the full test project before merging.
