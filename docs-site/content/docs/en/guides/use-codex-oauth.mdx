The Codex OAuth provider uses a dedicated login flow for the ChatGPT Codex backend. Use it when you want CodexSwitch to route through the authenticated Codex backend instead of a normal API-key provider.

## Quick setup

1. Select the Codex OAuth provider.
2. Click the Codex OAuth login action in the desktop UI.
3. Complete the browser login flow.
4. Confirm the account appears in the provider state.
5. Set the provider as active.

## Multi-account behavior

CodexSwitch stores OAuth account metadata locally. Treat the app config directory as sensitive because OAuth-backed state can be as important as an API key.
