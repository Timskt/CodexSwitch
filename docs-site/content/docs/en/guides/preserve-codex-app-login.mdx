---
title: Preserve Codex App Login
description: Keep ChatGPT-login-only Codex App plugin features while using the local proxy.
icon: ShieldCheck
---

If you need Codex App plugins, sign in to Codex App with ChatGPT first, then enable the auth preservation setting before applying proxy configuration.

## Setting

```text
Settings > Auth > Preserve Codex App ChatGPT login for plugins
```

## Runtime behavior

In this mode, CodexSwitch still writes `config.toml`, but leaves the original `auth.json` login state in place.

Use this mode when plugin features depend on the original ChatGPT login and you still want Codex traffic routed through the local proxy.
