Use `AnthropicMessages` when the upstream service accepts Anthropic Messages-style requests.

## Quick setup

1. Create or choose the Anthropic Messages provider.
2. Set protocol to `AnthropicMessages`.
3. Add the provider API key.
4. Configure the default Claude-compatible model.
5. Test a basic prompt before testing tool calls or reasoning behavior.

## What CodexSwitch maps

- Message payloads.
- Thinking or reasoning fields where supported.
- Tool call structures where supported.
- Usage normalization for local logging.

For Claude Code workflows, continue with [Use Claude Code](/en/docs/guides/use-claude-code).
