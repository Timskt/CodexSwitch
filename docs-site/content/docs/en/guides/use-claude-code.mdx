CodexSwitch can write managed Claude Code settings so Claude Code talks to the local proxy.

## Managed file

```text
%USERPROFILE%\.claude\settings.json
```

## Setup path

1. Configure a Claude-compatible or Anthropic Messages provider.
2. Select the active provider for Claude Code.
3. Keep the local proxy enabled.
4. Let CodexSwitch write managed Claude Code settings.
5. Verify Claude Code can reach the local base URL and model.

## Restore behavior

Original settings are preserved as `settings.json.bak` while managed config is active and restored when the proxy is stopped.
