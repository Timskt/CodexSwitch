---
title: Managed Client Config
description: Reversible writes to Codex and Claude Code local configuration files.
icon: FileCog
---

Managed client configuration is deliberately reversible. CodexSwitch writes local client config while the proxy is enabled and preserves originals with `.bak` backups.

## Managed files

| File | Purpose |
| --- | --- |
| `%USERPROFILE%\\.codex\\config.toml` | Managed Codex endpoint configuration. |
| `%USERPROFILE%\\.codex\\auth.json` | Managed Codex auth, unless Codex App auth preservation is enabled. |
| `%USERPROFILE%\\.claude\\settings.json` | Managed Claude Code base URL, token, and model settings. |

## Safety rule

Treat these files as sensitive. They can contain API keys, OAuth tokens, and local routing state.
