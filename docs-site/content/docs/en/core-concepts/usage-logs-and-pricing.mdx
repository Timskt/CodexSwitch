CodexSwitch records request usage locally and estimates cost from an editable model pricing catalog.

## Local files

| File | Purpose |
| --- | --- |
| `%APPDATA%\\CodexSwitch\\model-pricing.json` | Editable pricing catalog for estimates. |
| `%APPDATA%\\CodexSwitch\\usage-logs\\yyyy\\MM\\usage-yyyy-MM-dd.jsonl` | Partitioned usage logs. |

## Dashboard views

The usage dashboard tracks request logs, provider statistics, model statistics, 24-hour, 7-day, and 30-day trends.

Cost estimates depend on the local pricing catalog and should be treated as estimates, not billing statements.
