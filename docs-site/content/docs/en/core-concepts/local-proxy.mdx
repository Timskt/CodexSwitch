`ProxyHostService` owns the local Kestrel server. It binds to the configured host and port, maps local routes, applies managed Codex and Claude Code client configuration when running, and restores backups when stopped or disabled.

## Public local routes

| Route | Purpose |
| --- | --- |
| `/health` | Health check for the local proxy process. |
| `/v1/models` | Lists models visible through the active provider configuration. |
| `/v1/responses` | Main OpenAI Responses-style inbound route. |
| `/v1/messages` | Claude Code / Anthropic-style local route where supported. |

## Runtime behavior

- Keep the host on `127.0.0.1` unless you intentionally expose the proxy.
- Provider resolution happens per request.
- Usage records are written after response usage can be normalized.
- Managed local files are restored when the proxy is disabled.
