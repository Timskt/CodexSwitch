The `CodexSwitchUI` submodule contains reusable Avalonia controls, tokens, themes, and an existing desktop docs gallery.

The gallery categories seed this documentation hub:

- Overview
- Tokens
- Layout
- Forms
- Navigation
- Overlay
- Feedback
- Data Display
- Utilities
- Design Review

This Fumadocs site does not replace the Avalonia gallery. It documents the design system at a conceptual and contributor level.

Layout coverage now includes shadcn-style Sidebar Provider, Sidebar, Trigger, Rail, and Inset contracts alongside Resizable panel group anatomy for panels, handles, grips, tracks, orientation, and constraints. Sidebar trigger and rail commands now update `can-toggle` and `command-blocked` before changing open state, and the same per-example inline AXAML source workflow is used by the Avalonia Docs gallery.

Primitive coverage now includes a Direction provider that maps Web `ltr` / `rtl` state to Avalonia `FlowDirection`, documents nested direction islands, and keeps each rendered Docs case paired with its own inline AXAML source toggle.
