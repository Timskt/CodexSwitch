The `CodexSwitchUI` submodule contains reusable Avalonia controls, tokens, themes, and an existing desktop docs gallery.

The gallery categories seed this documentation hub:

- Overview
- Tokens
- Forms
- Navigation
- Overlay
- Feedback
- Data Display
- Utilities
- Design Review

This Fumadocs site does not replace the Avalonia gallery. It documents the design system at a conceptual and contributor level.
