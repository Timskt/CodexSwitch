Navigation components organize the desktop app shell and component gallery.

## Component families

- Tabs for root-owned `SelectedValue`, source-aware `ValueChanged` parity with Web `onValueChange`, automatic/manual activation, loop boundaries, and horizontal/vertical roving triggers
- Breadcrumb with Root/List/Item/Link/Page/Separator/Ellipsis anatomy, root `LinkActivated`, href and source metadata, primary pointer release vs keyboard/programmatic activation, current-page suppression, and dropdown composition
- Navigation menu with Root/List/Item/Trigger/Content/Link/Indicator/Viewport anatomy, controlled `ActiveValue`, active-item change events, top-level link commands, and link activation events
- Menubar with Root/Trigger/Content anatomy, persistent trigger menus, `ItemSelected` leaf metadata, checkbox/radio items, submenu handoff, and loading suppression
- Accordion with `ValueChanged` parity for Web `onValueChange`, changed item/value/index metadata, trigger/keyboard/programmatic sources, single/multiple modes, and roving trigger keys
- Collapsible with measured content height animation, trigger click/Enter/Space toggling, source-aware `OpenChanged`, and reduced-motion final states
- Dropdown button with trigger-owned primary pointer release, Enter/Space/ArrowDown open parity, source-aware `OpenChanged`, Escape focus return, close-on-select child buttons, and leaf menu item select dismissal
- Split button
- Menu and context menu with Web `onSelect`-style `ItemSelected` metadata, pointer/keyboard/programmatic sources, checkbox/radio state, submenus, shortcuts, groups, and close-on-select behavior
- Command with search text filtering, command separators, shortcut slots, scrollable lists, loop navigation, command-blocked item suppression, and source-aware item-selected events for pointer, keyboard, and programmatic selection
- Kbd and KbdGroup for command hints, button trailing shortcuts, tooltip help, and input group addons
- Side nav with root `SelectedValue`, item `Value`, source-aware `ValueChanged` parity with Web `onValueChange`, primary pointer release vs keyboard activation metadata, command-blocked row suppression, and legacy sibling-selection fallback
- Segmented control with root `SelectedValue`, source-aware `ValueChanged` parity with Web `onValueChange`, primary pointer release vs keyboard activation metadata, command-backed controlled segments, command-blocked state classes, and measured moving indicator
