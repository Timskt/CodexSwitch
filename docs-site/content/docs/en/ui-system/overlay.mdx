Overlay components handle provider editors, alert dialogs for destructive confirmations, filter sheets, Vaul-style drawers, command dialogs, popovers, hover previews, and focus management.

## Review states

- Open and closed.
- Dialog root/trigger/overlay/content/close composition, source-aware `OpenChanged`, modal scrim, Enter/Space trigger toggling, Escape/outside dismissal, and focus return.
- Alert dialog trigger/content composition, source-aware `OpenChanged`, cancel/action paths, least-destructive focus, modal scrim, and required-response outside pointer policy.
- Sheet trigger/content composition, inherited source-aware `OpenChanged`, modal scrim, right/left/top/bottom side placement, edge motion, close control, and focus return.
- Drawer trigger/content composition, inherited source-aware `OpenChanged`, modal scrim, directions, handle drag threshold/release, drag-dismiss-ready state, scrollable body, sticky footer actions, close controls, and focus return.
- Command dialog trigger/content composition, inherited source-aware `OpenChanged`, input forwarding, search filtering, separators, shortcuts, close-on-select, loading suppression, source-aware item-selected events, and focus return.
- Popover trigger/content/close/arrow composition, source-aware `OpenChanged`, Enter/Space trigger toggling, side/align motion, Escape/outside dismissal, and focus return.
- Tooltip provider/trigger/content composition, hover delay, immediate focus open, source-aware `OpenChanged`, Escape/Enter/Space dismissal, side-aware arrows, keyboard shortcut content, and disabled trigger wrappers.
- Hover card trigger/content composition, open-change events, open/close delays, side/align placement, arrow visibility, rich preview content, and disabled-root suppression.
- Focus trap.
- Escape, outside-click, close-on-select, drag-release, and focus-return behavior.
- Destructive confirmation copy.
- Theme contrast in light and dark modes.
